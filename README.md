# 🐍 Snake Game

A classic Snake Game built using HTML, CSS, and JavaScript with a simple and interactive design.

## 🚀 About the Game

This project recreates the popular Snake Game where the player controls a snake, collects food, and tries to achieve the highest score without colliding with walls or itself.

The game was developed to practice JavaScript logic, DOM manipulation, and game development fundamentals.

## ✨ Features

- Classic snake gameplay
- Score tracking system
- Smooth movement controls
- Game over detection
- Responsive and simple UI
- Keyboard controls

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Project Goal

The goal of this project is to improve problem-solving skills and gain practical experience in JavaScript game development and interactive web applications.
